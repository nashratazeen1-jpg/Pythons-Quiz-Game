# Pythons-Quiz-Game
In this Quiz Game, ***it just Asks User Simple 5 Basic Questions of Python with Option A &amp; Option B***. If User Enters the Correct Option then he will get 1 point as a Score, if he enters the Wrong Options Then he get no +1 Point and After Completing all 5 Questions the Final Score of User will be displayed along with the You WINS or You LOOSE Label.

# 🎮Game WorkFlow:

1. First, **Questions will Display to User** with *Option A & Option B*.
2. Now, ***User*** will ***Enter the Option***.
3. If, the *Entered Option is Correct*, **1 will be added to User's Score**.
4. Else, the **User Enters** the **Wrong Option +1 won't Added, Original Score will Remain As it Ease**.
5. At the End, **After Answering All Questions Final Score** and *WINS/LOOSE* will be Displayed.

# 🚀KeyPoints Learned:

1. Implementation of if-else Conditonal Statements.
2. Usage of **import module, Conditional & Arithemetic Operators**.
3. Understand about ***Escape Sequence -> \n  & \t***.
4. Hands On Practice of Python's Concepts on Usefull Mini Games.
5. Learned Basic Proficiency in Python.
